## CLI
Command Line Interface
命令交互

[GitHub](https://github.com/aws/aws-cli)

### 安装
* [安装手顺](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html)
* 查看版本：
  * cmd
  * aws --version
* 创建访问密钥
* 配置config
    `aws configure`
    输入
    访问密钥的用户名
    密码
    区域，选择近的：ap-northeast-2                 Seoul

### CloudShell（云Linux系统）
    网页访问
  * aws --version
    区域 ap-south-1                     Mumbai

### 开始使用
`aws iam list-isers`
