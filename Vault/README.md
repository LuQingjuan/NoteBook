## [效率爆表！分享一套完整的个人工具流+高效笔记方式](https://zhuanlan.zhihu.com/p/440575138)
* 用者无机心
**机心**指的是「机巧功利之心」，说的是我们太过于依赖工具去提高「效率」，结果到头来却忽略了「效能」
**效率**指的是正确地做事儿
**效能**指的是做正确的事儿
应该摒弃**机心**，先完成再完美

* 工具选择三条原则：
  1. **如无必要，勿增实体**：只有当你真正有这样的需求时才会去思考我该怎么通过增加工具来优化我的知识生产方式。
  2. **跨平台与同步性**
  3. **数据安全**
