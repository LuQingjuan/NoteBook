# VS Code
## 快捷键
* 折叠: `Ctrl`+`K`然后`Ctrl`+`0`
* 展开: `Ctrl`+`K`然后`Ctrl`+`J`
* 折叠代码段: `Ctrl`+`Shift`+`[`
* 展开代码段: `Ctrl`+`Shift`+`]`
* 格式化代码: `Ctrl`+`K`然后`Ctrl`+`F`
* 检索: `Ctrl`+`F`
* 全局检索: `Ctrl`+`Shift`+`F`
* 命令面板: `Ctrl`+`Shift`+`P`

* 切换到资源管理器: `Ctrl`+`Shift`+`E`
* 切换为文件: `Ctrl`+`Tab`
* 跳转到某行: `Ctrl`+`G`
* 打开终端: `Ctrl`+`~`
